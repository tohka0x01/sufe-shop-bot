[Important Notes]
=================
1. File format: Plain text file (.txt) only
2. Encoding: UTF-8
3. Single-line codes: One code per line
4. Multi-line codes:
   - No empty lines within a single code
   - Separate different codes with empty lines
   - Separator lines (=== or ---) are treated as empty lines
5. Special characters supported: @ # $ % & * - _ + = : ; , . / \ |
6. Maximum file size: 10MB recommended
